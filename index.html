Team Name / Mission

Our team name is Rubber Ducky Squad and our mission is to provide high-quality code on time for our customers

Problem Statement

Our problem for this application will be to develop a two-sided application that will have functionality to reserve rooms for a hotel. This application will have an employee facing part, that will be used to do basic registration and reservation for customers but also have administrator functionality. The other side of the application will be a customer-facing side, which will allow customers to visit the application and register and pay for the rooms.
